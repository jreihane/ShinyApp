Linear Regression with mtcars dataset

Where the idea came from?

  • In this project we are asked to make a Shiny application and it's documents.
  • I was inspired by the "Regression Models"! Because I enjoyed that course a lot!
  • So I created an application that helps users to make linear regression plots dynamically.

Linear Regression with mtcars dataset

Introduction to my application

  • The idea is very simple!
  • You have a dataset ready for you and a linear regression model.
  • The output is fixed: "mpg" variable is the output.
  • You select each variable you want to add to the model.
  • So all variables are listed in the page as checkboxes.

Linear Regression with mtcars dataset

How it works?

  • There are two tab in this app.
  • One for application itself.
  • And one that brought you here to see these slides.
  • In App tab, There are two parts.
  • First for creating the model
  • And the other for showing plots. -Note: It was possible to use other form of layouts, but I found it more interesting to have a sidebar and mainPanel being vertically aligned ... My bad!!

Linear Regression with mtcars dataset

How it works? ... continued

  • In sidebar there is a checkbox group that shows variable names
  • Each time you check/uncheck a variable name, the formula will be updated.
  • Then you can see 4 plots, regarding to the linear regression model you have fit.
